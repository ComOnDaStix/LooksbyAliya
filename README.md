# Looks by Aliya

A concept marketing site for **Looks by Aliya** — a Milwaukee-based hair stylist and makeup artist specializing in balayage, color corrections, blonding, bridal hair, and glam.

Single-page static site (`index.html`) — no build step required.

## Sections
Hero · About · Services · Pricing menu · Portfolio · Reviews · Booking · Footer

## Local preview
```bash
node serve.mjs        # serves the project root at http://localhost:3000
```

## Deploy
Static site — deploys to Vercel with zero configuration (output is `index.html` at the repo root).

---
Real business details (services, pricing, hours, address) sourced from looksbyaliya.glossgenius.com. Testimonials are sample placeholders. Imagery is from the Barb template and is for mockup purposes.
